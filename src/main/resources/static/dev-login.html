Trego - Login Firebase (DEV)

Página solo para pruebas locales. Login real con Google -> idToken de Firebase -> JWT de Trego para usar en Swagger.

1Configurar Firebase Web SDK

Si no hay una Web App registrada en Firebase, registrala en Console -> Project Settings -> Your apps -> Web. Copiá el objeto firebaseConfig y pegalo abajo. El config se guarda en localStorage para no tener que pegarlo cada vez.

2Login con Google

3Canjear por JWT de Trego

Envia el idToken a POST /api/auth/google.

4Usar en Swagger

Abri /swagger-ui/index.html, click en el boton Authorize, pega el JWT (sin el prefijo Bearer), Authorize -> Close.
